# HEIC to JPG Converter

Simple drag-and-drop Windows app to batch convert `.heic` / `.heif` photos to `.jpg`.

## Download

Go to **[Actions](../../actions)** → latest run → download **HEIC-to-JPG-Windows** artifact.

## Usage

1. Open `HEIC-to-JPG.exe`
2. Click the drop zone to select `.heic` files
3. Choose an output folder
4. Click **Convert**
